📦 Amazon UI Clone
A feature-rich, pixel-perfect frontend recreation of the Amazon web ecosystem. This project demonstrates advanced CSS techniques including Flexbox, CSS Grid, and Keyframe Animations to provide a seamless shopping and streaming UI experience.

🌟 Key Features
Dynamic Homepage: Features a high-fidelity navigation bar with hover states, a multi-image hero slider using CSS animations, and a responsive product grid.

Prime Video Hub: A dedicated dark-themed portal for streaming content with a specialized layout for movie cards.

Authentication Flows: Custom-styled sign-in pages tailored for both the main Amazon store and the Prime Video branding.

Today's Deals: A clean, itemized list of discounted products with quick navigation back to the main store.

Responsive Layouts: Hand-coded layouts designed to maintain structure across various browser dimensions.

🛠️ Built With
HTML5: Semantic markup for SEO and accessibility.

CSS3: * Flexbox for complex navigation and product alignments.

@keyframes for the automated hero image transitions.

Transitions for interactive button and link hover effects.

Font Awesome: Professional iconography for the search, location, and cart elements.
